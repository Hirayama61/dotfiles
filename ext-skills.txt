# 外部 agent skill を宣言。mise run skills:sync で ghq clone & ~/.claude/skills へ symlink。
# 形式(コロンの有無とセグメント数で 3 形態を判定):
#   A. owner/repo:subdir            subdir 直下の各 skill を複数 symlink
#   B. owner/repo/path/to/skill     その 1 ディレクトリだけを単体 symlink(>=3 セグメント・コロンなし)
#   C. owner/repo                   subdir 既定 skills で複数 symlink
# ghq get の対象はいずれも先頭 2 セグメント owner/repo。
# symlink は再配布ではない(本体は ghq clone でローカルに置く)ため LICENSE 同梱は不要。
